Usando la definición de la suma de Riemann calcularemos el área bajo la curva de las siguientes funciones.

Para calcular el área de la 1ra funcion f(x)=100 - (x - 10)^4 + 50(x + 10)^2;[3,17] use el comando:

    make funcion1 N="Número de Hilos"


Para calcular el área de la 2da funcion f(x)=(x - 1)^2 + 25;[5,20] use el comando:

    make funcion2 N="Número de Hilos"


Para calcular el área de la 3ra funcion f(x)=(x - 4)^3 + 10;[2,16] use el comando:

    make funcion3 N="Número de Hilos"


Para calcular el área de las 3 funciones use el comando:

    make all N="Número de Hilos"
